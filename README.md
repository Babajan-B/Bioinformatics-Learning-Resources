### Prerequisites by Level

**Beginner:**
- Basic molecular biology (DNA, RNA, proteins, central dogma)
- High school mathematics
- Comfort with computers and file systems
- Willingness to learn command line

**Intermediate:**
- University-level biology or biochemistry
- Basic programming in Python or R
- Fundamental statistics (hypothesis testing, distributions)
- Linux command line proficiency

**Advanced:**
- Graduate-level molecular biology/genomics
- Proficient programming in multiple languages
- Statistical modeling and machine learning
- Experience with HPC environments

## ğŸ’» Programming for Bioinformatics

### Python

#### Essential Libraries

```python
# Core Scientific Computing
numpy              # Numerical arrays and operations
pandas             # Data manipulation and analysis
scipy              # Scientific computing (stats, optimization)

# Visualization
matplotlib         # Publication-quality plots
seaborn            # Statistical visualization
plotly             # Interactive visualizations

# Bioinformatics-Specific
biopython          # Biological computation toolkit
pysam              # SAM/BAM file handling
pyvcf / cyvcf2     # VCF file parsing
pybedtools         # BED file operations

# Single-Cell Analysis
scanpy             # Single-cell analysis in Python
anndata            # Annotated data matrices
```

### R & Bioconductor

#### Essential Packages

```r
# Core Tidyverse
tidyverse          # Meta-package: dplyr, ggplot2, tidyr, etc.

# Bioconductor Core
BiocManager        # Package installation
GenomicRanges      # Genomic intervals
SummarizedExperiment  # Feature x sample data

# RNA-Seq Analysis
DESeq2             # Differential expression
edgeR              # Differential expression
limma              # Linear models

# Single-Cell Analysis
Seurat             # Comprehensive scRNA-seq toolkit
SingleCellExperiment  # Data structure
Monocle3           # Trajectory analysis
```

### Bash/Linux

#### Essential Commands

```bash
# FASTQ processing
grep -c "^@" file.fastq        # Count reads
awk 'NR%4==2' file.fastq       # Extract sequences

# SAM/BAM operations
samtools view -c file.bam       # Count alignments
samtools flagstat file.bam      # Alignment statistics

# Parallel processing
parallel -j 8 'gzip {}' ::: *.fastq
```
